# سبق کا تعارف

آئیے **Docusaurus** کو **5 منٹ سے بھی کم** میں دریافت کریں۔

## شروعات کریں

**نئی سائٹ بنا کر** شروع کریں۔

یا **[docusaurus.new](https://docusaurus.new)** کے ساتھ **Docusaurus کو فوری طور پر آزمائیں**۔

### آپ کو کیا چاہیے

- [Node.js](https://nodejs.org/en/download/) ورژن 20.0 یا اس سے اوپر:
  - Node.js انسٹال کرتے وقت، آپ کو تمام انحصارات سے متعلق چیک باکسز کو چیک کرنے کی سفارش کی جاتی ہے۔

## نئی سائٹ بنائیں

**کلاسک ٹیمپلیٹ** استعمال کرتے ہوئے نئی Docusaurus سائٹ بنائیں۔

کمانڈ چلانے کے بعد کلاسک ٹیمپلیٹ خودکار طور پر آپ کے پروجیکٹ میں شامل ہو جائے گی:
```bash
npm init docusaurus@latest my-website classic
```

آپ یہ کمانڈ Command Prompt، Powershell، Terminal، یا اپنے کوڈ ایڈیٹر کے کسی بھی مربوط ٹرمینل میں ٹائپ کر سکتے ہیں۔

یہ کمانڈ Docusaurus چلانے کے لیے ضروری تمام انحصارات بھی انسٹال کرتی ہے۔

## اپنی سائٹ شروع کریں

ڈیولپمنٹ سرور چلائیں:
```bash
cd my-website
npm run start
```

`cd` کمانڈ اس ڈائرکٹری کو تبدیل کرتی ہے جس کے ساتھ آپ کام کر رہے ہیں۔ اپنی نئی بنائی گئی Docusaurus سائٹ کے ساتھ کام کرنے کے لیے، آپ کو ٹرمینل کو وہاں نیویگیٹ کرنا ہوگا۔

`npm run start` کمانڈ آپ کی ویب سائٹ کو مقامی طور پر بناتی ہے اور اسے ڈیولپمنٹ سرور کے ذریعے پیش کرتی ہے، جو http://localhost:3000/ پر دیکھنے کے لیے تیار ہے۔

`docs/intro.md` (یہ صفحہ) کھولیں اور کچھ لائنیں ایڈٹ کریں: سائٹ **خودکار طور پر دوبارہ لوڈ** ہوتی ہے اور آپ کی تبدیلیاں ظاہر کرتی ہے۔
